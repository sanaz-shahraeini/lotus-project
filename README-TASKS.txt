1- Install this executable file (Redis-x64-3.0.504.msi)

NOTE: IF YOU CAN'T INSTALL, YOU CAN GET THE FILE FROM THIS LINK -> https://github.com/microsoftarchive/redis/releases

2- Make sure you checked Add environment option while installing

3- Write port on somewhere that showed in you installing process

4- Open up CMD and type redis-cli ping
   + If Redis sent you back "PONG", your Redis server is running and working

5- Make sure your port that you installed and wrote it and your port in your settings.py project is the same


6- Run the project
